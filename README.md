# Optimisasi Website STIKes Mitra RIA Husada Jakarta

**Optimisasi Website STIKes Mitra RIA Husada Jakarta** merupakan proyek peningkatan tampilan pada situs resmi kampus yang sebelumnya beralamat di **mrhj.ac.id**. Proses optimisasi ini mencakup perbaikan desain, navigasi, responsivitas, serta penyesuaian konten agar lebih modern, informatif, dan user-friendly.

Proyek ini telah menghasilkan tampilan baru yang lebih bersih.
